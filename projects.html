Projects Completed

Throughout my academic and personal learning journey, I have worked on several impactful projects that reflect my skills in software development, automation, and system design:

  1. Network Automation Using Ansible
    This project involved automating network device configurations using Ansible, an open-source automation tool. I developed playbooks to manage and configure network infrastructure efficiently, reducing manual errors and improving deployment speed. This experience deepened my understanding of network management, scripting, and infrastructure-as-code principles.
  2. Blood Donation System
    I designed and developed a web-based Blood Donation Management System aimed at connecting donors with recipients and healthcare facilities. The system allows for donor registration, blood requests, and inventory tracking. I used PHP, JavaScript, and MySQL to build a functional and user-friendly platform that improves the efficiency and accessibility of blood donation services.
  3. Graduation Certificate Authentication System
    This project focused on developing a system for verifying academic certificates to prevent fraud and ensure authenticity. The solution uses secure technologies and a web interface to allow institutions and employers to validate certificates issued by educational bodies. It strengthens trust and transparency in academic credentials, and showcases my skills in system design, web development, and data security.
